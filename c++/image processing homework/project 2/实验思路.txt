二、二值化部分
1.读入一个BMP
2.记录最小和最大的Y（整体）
3.遍历像素，遍历TH，记录N1，N2，μ1，μ2.
4.计算TH对应的σ，比较得到最终整体阈值TH0.
5.遍历左上角位置(0 ~ height - WindowSize, 0 ~ width - WindowSize).
(1)遍历像素，找出窗口内最大Y和最小Y
(2)遍历像素，遍历TH，记录N1，N2，μ1，μ2.
(3)计算σ，得到子块的阈值THx
(4)遍历像素，将THx和自有的TH做平均。
6.遍历像素，TH = k TH0 + (1-k) THx.判断黑白，进行二值化。
7.输出图像
三、膨胀收缩部分
1.建立一个采样矩阵，矩阵为奇数长，有中心
2.新建两个新的PixelVec，存储输出结果。（使用new，存在堆中）
3.遍历采样矩阵的中心
(1)遍历矩阵元素，将相应位置做与逻辑，所有结果做或逻辑，得到dilation
(2)遍历矩阵元素，将相应位置做取反的与逻辑，所有结果做或逻辑，得到erosion
4.输出图片
四、合拢张开部分
1.沿用PixelVec
2.对上述两个vec进行相反操作
3.输出图片
